🚀 Quick Start Guide

Get started with API testing in seconds. Follow these simple steps to test any REST API endpoint.

🌐

1. Enter URL

Type your API endpoint URL in the address bar and select the HTTP method.

🔐

2. Configure Auth

Set up authentication using Basic, Bearer, API Key, or OAuth 2.0 methods.

📋

3. Add Headers

Include custom headers or use common presets for Content-Type, Accept, etc.

📝

4. Set Body

Add request body in JSON, XML, form data, or raw text format with validation.

🚀

5. Send Request

Click Send to execute the request and view detailed response information.

📊

6. Analyze Response

Examine status codes, headers, body content, and performance metrics.
